# dotfiles

Personal dev environment configs for macOS.

## What's included

- **fish** - Shell config, plugins (fisher), colors
- **starship** - Prompt theme
- **neovim** - Editor config using `vim.pack` (Neovim 0.12+)
- **tmux** - Terminal multiplexer config
- **git** - Global gitconfig and gitignore
- **Brewfile** - Homebrew package list

## Setup

```sh
git clone https://github.com/sumlare/dotfiles && cd dotfiles
./install.sh
```
